提示词：
**角色设定：**  
你是一位世界级的 Creative Technologist，精通 Three.js (WebGL)、GLSL 着色器逻辑以及 Google MediaPipe 计算机视觉集成。  
**任务目标：**  
请编写一个单文件 (Single-File) 的 HTML 项目，文件名为 **`春节快乐`**。该项目必须包含完整的 HTML 结构、CSS 样式和基于 ES Module 的 JavaScript 代码。  
**核心技术栈约束 (必须严格遵守)：**  
依赖管理： 使用 `<script type="importmap">` 引入：
three:  https://cdn.jsdelivr.net/npm/three@0.160.0/build/three.module.js 
three/addons/:  https://cdn.jsdelivr.net/npm/three@0.160.0/examples/jsm/ 
@mediapipe/tasks-vision:  https://cdn.jsdelivr.net/npm/@mediapipe/tasks-vision@0.10.3/+esm 
代码风格： 使用现代 ES6+ 语法，类 (Class) 结构管理粒子，异步/等待 (Async/Await) 处理加载。
**📝 详细开发规格说明书：**  
1. **UI 设计与 CSS 样式 (Visual Identity)**  
   - **字体：** 引入 Google Fonts 'Ma Shan Zheng' (标题，支持中文书法风格) 和 'Noto Sans SC' (正文，中文字体)。  
   - **配色：** 背景 `#0a0a0a`（深空夜色）。主色调：中国红 `#e60012`，鎏金 `#d4af37`，吉祥白 `#fff9f0`。  
   - **UI 组件：**  
     - **Loader：** 全屏遮罩中间显示 **40px 金色旋转灯笼**（替代_spinner_），下方文字 `"LOADING LUNAR MAGIC"`。  
     - **标题：** 顶部居中 `<h1>` **"欢度春节"**，字号 56px，文字渐变（红→金），辉光阴影模拟灯笼光晕。  
     - **上传控件：** 按钮文字改为 **"上传团圆照"**，悬停时显示金色粒子飘落动画。  
     - **提示文本：** 按钮下方显示 **"按 'H' 隐藏界面 | 挥手切换场景"**。  

2. **Three.js 场景构建 (Scene Setup)**  
   - **环境 (Environment)：**  
     - 背景替换为 **动态宣纸纹理**（Canvas 生成水墨晕染效果），叠加飘落的金色 **"福" 字粒子**（每 5 秒随机生成 1 个）。  
   - **灯光系统：**  
     - 关键修改：SpotLight (红色, 强度 1000, 位置 30,40,40) 模拟 **灯笼暖光**；SpotLight (青金色, 强度 500, 位置 -30,20,-30) 模拟 **月光清辉**。  

3. **粒子系统与内容 (Content Generation)**  
   - **粒子总数：** 1500 个主体粒子 + 2500 个尘埃粒子（尘埃替换为 **金色光点**，模拟爆竹余烬）。  
   - **几何体混合：**  
     - **传统窗花造型**：使用 Canvas 2D 绘制镂空剪纸纹样（梅花/锦鲤），创建 CanvasTexture。  
     - **红绸灯笼**：MeshPhysicalMaterial（红色带 clearcoat 模拟丝绸光泽），顶部添加金色流苏模型。  
     - **爆竹串**。TubeGeometry 沿 CatmullRomCurve3 生成弯曲竹枝，Canvas 2D 生成红纸金纹贴图。  
   - **照片墙功能：**  
     - 默认纹理文字改为 **"新春快乐"**（毛笔字体，金色描边）。  
     - 相框为 **雕花木框**（BoxGeometry + 木质凹凸贴图），四角悬挂微型 **中国结粒子**。  

4. **状态机与动画逻辑 (State Machine & Logic)**  
   - **Mode 1: LANTERN TOWER (灯笼塔模式)**  
     粒子排列为 **悬浮灯笼阵列**（螺旋圆柱体结构）。公式：`radius = 8 + Math.sin(t * 15) * 3`（模拟摇曳光效）。  
   - **Mode 3: FOCUS (聚焦模式)**  
     聚焦目标外的粒子自动组成 **"福" 字背景墙**（平面网格+粒子位移）。  

5. **MediaPipe 计算机视觉集成 (Computer Vision)**  
   - **手势映射优化：**  
     - **拱手礼识别**：双手指尖距离 < 0.15 且手腕高度 > 0.7 → 触发 **红包雨特效**（新增粒子系统）。  
     - **舞狮头动作**：快速上下摆动（手腕 Y 轴加速度 > 0.3）→ 场景播放 3D 狮头动画（预加载 GLB 模型）。  
   - **交互映射：**  
     手掌中心坐标额外控制 **灯笼亮度**（rotation.y → 环境光强度）和 **宣纸背景水墨浓度**（rotation.x → 背景透明度）。  



请输出完整的、包含所有这些细节的单文件 HTML 代码。
